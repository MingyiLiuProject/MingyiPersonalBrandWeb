# Mingyi Personal Brand Web

A Next.js personal brand site for Mingyi Liu: product work, research projects,
future tools, and a small build lab.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- pnpm
- Vercel-ready deployment

## Local Development

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Useful Commands

```bash
pnpm lint
pnpm build
```

## First Content Areas

- `/` brand homepage
- `/work` product and research case studies
- `/build` future tools and prototypes
- `/about` personal story
- `/contact` collaboration entry point
